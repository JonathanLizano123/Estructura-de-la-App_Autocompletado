# Estructura-de-la-App_Autocompletado
Crear una carpeta .streamlit y poner ahi el archivo config.toml y los demas archivos quedan afuera
Reemplazar los archivos que dice pon_... por los generados con los scripts

# Instalar las librerias
pip install streamlit tensorflow numpy

# Y correr con: 
streamlit run app.py
